# 🤖 ai-resources

Recursos, configuraciones y scripts relacionados con inteligencia artificial. Enfocado en automatización con APIs de IA y herramientas de agentes autónomos como OpenClaw, desde una perspectiva de sysadmin.

---

## 📁 Contenido

```
ai-resources/
├── prompts/        # Prompts útiles organizados por caso de uso
├── automation/     # Scripts que combinan IA con tareas de sistemas
├── openclaw/       # Configuraciones y skills para OpenClaw
└── api-examples/   # Ejemplos de uso de las APIs de OpenAI y Anthropic
```

---

## 🎯 Enfoque

No es un repositorio de modelos ni de fine-tuning. El objetivo es aplicar IA de forma práctica en el día a día: automatizar tareas, integrar agentes en flujos de trabajo y sacar partido a las APIs sin complicarse.

---

## 🛠️ Herramientas utilizadas

- OpenAI API
- Anthropic API (Claude)
- OpenClaw

---

## ⚠️ Notas

Las claves de API nunca se incluyen en el repositorio. Usa siempre variables de entorno o un archivo `.env` que no subas a Git.

```bash
# Ejemplo de .env (no subir a Git)
OPENAI_API_KEY=tu_clave_aqui
ANTHROPIC_API_KEY=tu_clave_aqui
```
